Css Selector

Class Selector

Description

The class selector in CSS is denoted by a dot (.) followed by the class name. It is used to select and style HTML elements with a specific class attribute

Class1,Class2 Selector

Description

The .class1.class2 selector in CSS is used to select elements that have both the class1 and class2 classes. It targets elements that have both specified classes in the HTML.

id Selector

Description

The #id selector in CSS is used to select and style a specific HTML element with a unique identifier, denoted by its id attribute.

Universal Selector

Description

The universal selector * in CSS selects all elements on a webpage.It matches any element type and is often used as a way to apply styles globally.

Element Selector

Description

Elements are defined by HTML tags, and each tag represents a specific type of content or structure within the document.

Element.class Selector

Description

The element.class selector in CSS is a shorthand notation for styling HTML elements with a specific class. It targets and applies styles to all elements of a particular HTML element type that have the specified class

Element,Element Selector

Description

The #id selector in CSS is used to select and style a specific HTML element with a unique identifier, denoted by its id attribute.

Element Element Selector

Description

The universal selector * in CSS selects all elements on a webpage. It matches any element type and is often used as a way to apply styles globally.

Element>Element Selector

Description

The "Element > Element" selector in CSS targets a specific element that is a direct child of another specified element.

Element+Element Selector

Description

The "Element + Element" selector in CSS, known as the adjacent sibling combinator, selects an element that is immediately preceded by a specified sibling element.

Element1~Element2 Selector

Description

The "Element1 ~ Element2" selector in CSS, known as the general sibling combinator, selects all "Element2" elements that are siblings and share the same parent as "Element1," regardless of their order.

[attribute]

Description

The [attribute] selector in CSS targets elements that have a specified attribute, regardless of its value.

[attribute=value] Selector

Description

The [attribute=value] selector in CSS targets elements with a specific attribute that has a matching value.

[attribute~=value] Selector

Description

The [attribute~=value] selector in CSS targets elements with an attribute that contains a specific space-separated value within its attribute value.

[attribute|=value] Selector

Description

The [attribute|=value] selector in CSS targets elements with an attribute that has a value equal to the specified value or begins with the specified value followed by a hyphen.

[attribute^=value] Selector

Description

The [attribute^=value] selector in CSS targets elements with an attribute that starts with a specified value.

[attribute$=value] Selector

Description

The [attribute$=value] selector in CSS targets elements with an attribute that ends with a specified value.

[attribute*=value] Selector

Description

The [attribute*=value] selector in CSS targets elements with an attribute that contains the specified value anywhere within its attribute value.

:active Selector

Description

The :active selector in CSS is a pseudo-class that selects and styles the active state of an element, typically when it is being clicked or activated by a user.

::after Selector

Description

The ::after selector in CSS is a pseudo-element that selects and styles the content that is generated after the content of a specified element.

::before Selector

Description

::before is a CSS pseudo-element that allows you to insert content before the actual content of an HTML element.

:checked Selector

Description

:checked is a CSS pseudo-class selector that targets and styles HTML input elements when they are in a checked or selected state.

::default Selector

Description

The ::default pseudo-element is used to select and style the distributed nodes that are not assigned to a specific slot.

:disabled Selector

Description

:disabled is a selector in CSS (Cascading Style Sheets) that targets and styles HTML elements that are currently disabled or inactive.

:empty Selector

Description

:empty is a CSS pseudo-class selector that targets and styles HTML elements that have no children or contain only empty text nodes.

:enabled Selector

Description

:enabled is a CSS pseudo-class selector that targets and styles HTML form elements, such as buttons or input fields, when they are in an enabled or active state.

:first-child Selector

Description

The :first-child selector is a CSS pseudo-class that targets and styles the first child element of its parent.

:first-letter

Description

The ::first-letter selector is a CSS pseudo-element that targets and styles the first letter of a block-level element.

:hover Selector

Description

The :hover selector is a CSS pseudo-class that targets and styles an HTML element when the user hovers over it with the cursor.

:in-range Selector

Description

The :in-range selector is a CSS pseudo-class that targets and styles HTML input elements with a range type (e.g., sliders) when their current value falls within the specified range.

:indeterminate Selector

Description

The :indeterminate selector is a CSS pseudo-class that targets and styles HTML input elements, typically checkboxes or radio buttons, when they are in an indeterminate state.

:invalid Selector

Description

The :invalid selector is a CSS pseudo-class that targets and styles HTML input elements with invalid input data, according to their specified validation constraint.

:lang(language) Selector

Description

The :lang(language) selector in CSS is used to target and style elements based on their language attribute, enabling customization of styles for specific languages on a webpage.

:last-child Selector

Description

The :last-child selector is a CSS pseudo-class that targets and styles the last child element of its parent. It allows you to apply styles specifically to the final child element within a container or parent element.

:last-of-type Selector

Description

The :last-of-type selector is a CSS pseudo-class that targets and styles the last element of a specific type among its siblings within a parent element.

:link Selector

Description

The :link selector is a CSS pseudo-class that targets and styles unvisited links in a document. It allows you to apply styles specifically to links that have not been clicked or visited by the user.

::marker Selector

Description

The ::marker selector in CSS styles the marker or bullet point of a list item, providing a way to customize its appearance in ordered or unordered lists.

:not Selector

Description

The :not selector in CSS targets and styles elements that do not match the specified selector inside the parentheses.

:nth-child(n) Selector

Description

The :nth-child(n) selector in CSS targets and styles the element that is the nth child of its parent.

:nth-last-child(n) Selector

Description

The :nth-last-child(n) selector in CSS targets and styles the element that is the nth child counting from the end of its parent.

:nth-last-of-type(n) Selector

Description

The :nth-last-of-type(n) selector in CSS targets and styles the element that is the nth child of a specific type counting from the end of its parent.

:nth-of-type Selector

Description

The :nth-of-type selector in CSS targets and styles the element that is the nth child of a specific type within its parent.

:only-of-type Selector

Description

The :only-of-type selector in CSS targets and styles an element that is the only child of its type within its parent.

:only-child Selector

Description

The :only-child selector in CSS targets and styles an element that is the only child of its parent.

:optional Selector

Description

The :optional selector targets and styles form elements that are not required, allowing customization of optional input fields.

:out-of-range Selector

Description

The :out-of-range selector targets and styles HTML input elements, such as sliders, when their current value is outside the specified range.

::placeholder Selector

Description

The ::placeholder selector in CSS targets and styles the placeholder text within an input or textarea element.

:read-only Selector

Description

The :read-only selector in CSS targets and styles HTML input elements that are in a read-only state, preventing user input.

:read-write Selector

Description

The :read-write selector in CSS targets and styles HTML input elements that are in a state allowing user input, as opposed to read-only.

:required Selector

Description

The :required selector in CSS targets and styles HTML input elements that have the "required" attribute set, indicating that user input is mandatory.

:root Selector

Description

TThe :root selector in CSS targets and styles the highest-level ancestor element in the document, often used to define global styles and variables.

::selection Selector

Description

The ::selection selector in CSS targets and styles the portion of text selected by the user, such as highlighting in a browser.

:target Selector

Description

The :target selector in CSS targets and styles the element that is currently the target of the document's URL fragment identifier, often used for styling active sections in a single-page application.

:valid Selector

Description

The :valid selector in CSS targets and styles HTML input elements with valid data according to their specified validation constraints, commonly used in conjunction with form validation

:visited Selector

Description

The :visited selector in CSS targets and styles HTML links that have been visited by the user, allowing customization of their appearance.
